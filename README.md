# dauphin
仿网易云播放器快速原型设计
dauphin为法语海豚之意，且软件基调为天空蓝，所以这简单寓意为倾听懂你的音乐软件


##################################
项目展示有ppt动态演示。
##################################



大三下学期阿里云产品经理校内实训，基于讲解内容，为期一周实现的仿网易云播放项目，基于Axure RP 8.0的快速模型。
使用的iphone6手机模型的尺寸429*881。原型中产品logo图标55*55。
